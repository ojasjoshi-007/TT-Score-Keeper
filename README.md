# 🏓 Table Tennis Score Keeper

A simple and interactive score tracking app built using HTML, CSS, and JavaScript.

## 🚀 Features
- Keep score for two players
- Set winning score dynamically
- Highlight winner and loser
- Display current server
- Add custom player names
- Reset game functionality

## 🛠️ Tech Used
- HTML
- CSS (Bulma)
- JavaScript (DOM Manipulation)
